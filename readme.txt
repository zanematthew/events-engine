== Description ==

zM Events & Venues is a WordPress plugin that allows anyone to creat an event, associate it with a venue, and allow users to create a schedule adding any event to their schedule on their self-hosted WordPress site.

[Learn more] or track the progress of [zM Events & Venues on Github](https://github.com/zanematthew/zm-events-venues/).

= Features =
* Each Event is associated to one and only one Venue.
* Each Event has the following meta information associated with it: start date, end date, fee.
* Events are grouped by: type, tags, or attendees.
* Each Venue can have multiple Events.
* Venues have the following meta information associated with it: email, phone, lat, long, city, state, zip, street, facebook link, and twitter link.
* Venues are grouped by: region, and tags.
* Be sure to check out the available extenstions!
